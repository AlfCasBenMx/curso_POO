2) Convertir Python a Excel

Prompt:
"Este código Python agrupa ventas por categoría y suma importes: <pega el snippet>. Explícalo en lenguaje de negocio y di qué columnas y fórmulas necesitaría en Excel para replicarlo."

Prompt:
"La fórmula propuesta es: <pega fórmula>. Indica 3 posibles fallos de entrada que la harían fallar y cómo mitigarlos (por ejemplo: formatos de fecha, valores nulos, separadores decimales)." 

4) Crear fórmula con condiciones escalonadas

Prompt:

Prompt:
"Explica en máximo 2 frases qué hace este script Python y por qué sería útil para un contador: <pega snippet>." 

Consejo de uso: siempre pide una explicación en lenguaje de negocio, ejemplos con datos y una verificación con un subconjunto pequeño.
