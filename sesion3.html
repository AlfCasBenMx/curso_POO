Sesión 3 — Laboratorio Python I

Instalación del entorno y primer script — preparar la máquina para los laboratorios.

Sesión C · Laboratorio 16-dic-25 · 15:00–16:00 (México)

Objetivo del laboratorio

Dejar listo el entorno de desarrollo en Windows para ejecutar los ejemplos del curso: instalar Python, configurar un entorno virtual, instalar dependencias y ejecutar scripts sencillos.

  • Verificar o instalar Python 3.
  • Instalar un editor recomendado (VS Code) y extensiones mínimas.
  • Crear y activar un entorno virtual.
  • Instalar dependencias con pip desde requirements.txt si aplica.
  • Ejecutar un primer script de prueba.

Agenda rápida (60 min)

  1. Comprobación previa y preguntas (10 min)
  2. Instalación y configuración (25 min)
  3. Primer script y pruebas (15 min)
  4. Preguntas y cierre (10 min)

Pasos prácticos

1) Verificar Python

Abre una ventana de cmd.exe y ejecuta:

python --version

2) Crear y activar un entorno virtual (Windows, cmd.exe)

python -m venv .venv
.venv\Scripts\activate

3) Actualizar pip e instalar dependencias

python -m pip install --upgrade pip
python -m pip install -r requirements.txt

4) Ejecutar un script de prueba

Crea un archivo hola.py con este contenido y ejecútalo:

print('Hola desde Python - Sesión 3')
python hola.py

Checklist rápido de preparación

  • Python 3 instalado y accesible desde la línea de comandos.
  • Editor instalado (VS Code recomendado).
  • Entorno virtual creado y activado.
  • Dependencias instaladas sin errores (si existe requirements.txt).
  • Primer script ejecutado correctamente.

Resolución de problemas comunes

  • Si python no se reconoce: instala Python y marca "Add to PATH" en el instalador.
  • Si la activación del entorno falla en PowerShell, ejecuta: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned en PowerShell (admin) o usa cmd.exe.
  • Errores al instalar paquetes: revisa la versión de pip y vuelve a intentarlo; revisa conexión a internet y permisos.
